# Financial-Analysis_Project
## Description
This project analyzes financial data (2022-2024) for Apple, Microsoft, and Tesla, including:
- Revenue trends
- Profitability metrics
- Asset-liability ratios
- Growth rate comparisons

## Files
- `FinancialAnalysis_Project.ipynb` - Jupyter Notebook with full code
- `Book2.csv` - Dataset used for analysis

## Requirements
- Python 3.x
- pandas
- numpy
- matplotlib
- seaborn
- jupyter

Install packages using:
```bash
pip install pandas numpy matplotlib seaborn jupyter
